# Frontend Aplikacije

## Najbitnije funkcionalnosti veb aplikacije

- Veb aplikacija namenjana je organizatorima događaja.
- Ogranizatori mogu da kreiraju događaj, kontrolišu pristup, upravljaju resursima i slično.
- Organizatori imaju potpuni uvid u prijavljene učesnike.
- Događaji mogu biti potpuno različiti.
- Veb aplikacija je potpuno funkcionalna (responzivna) i na desktop računarima i na mobilnim telefonima.

## Tehnologije

- React (JavaScript biblioteka)
- React Router (namenjen je za navigaciju između stranica)
- Axios (namenjen je za komunikaciju sa backendom)
- CSS framework (jos uvek odlučujemo)
- Node.js (namenjen za pokretanje lokalnog servera)

### Ukratko o React-u

- `React` je biblioteka kreirana za `JavaScript`.
- Koristi `komponentni pristup`, što znači da svaki deo korisničkog interfejsa predstavlja po jednu komponentu (npr. dugme, tabela...).

## Instalacija

### Kloniranje repozitorijuma

```bash
git clone http://gitlab.pmf.kg.ac.rs/si2025/vexa.git
cd src/frontend/eventplanner # Promena foldera
```

### NodeJS

#### Linux

```bash
sudo apt update 
sudo apt install nodejs npm
```

#### Windows / MacOS

```
Preuzeti sa zvaničnog sajta node: https://nodejs.org/en
```

### Instalacija potrebnih paketa

```bash
npm install
```

### Pokretanje aplikacije

```bash
npx expo start
```
