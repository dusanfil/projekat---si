# FRONTEND Aplikacije

## Kratak opis

- Predstavlja frontend aplikacije koji je povezan na ASP .NET Core backend.
- Omogućava registraciju i prijavu korisnika.

## Tehnologije

- **React** - https://reactjs.org/ - JavaScript biblioteka.
- **Bootstrap** - https://getbootstrap.com/ - služi za stilizovanje komponenti i responzivan dizajn.
- **React Router DOM** - https://reactrouter.com/ - služi za navigaciju između stranica

## Instalacija i pokretanje aplikacije

```bash
git clone http://gitlab.pmf.kg.ac.rs/si2025/vexa.git #Kloniranje repozitorijuma
cd sandbox/Marko_Mladenovic/demo/frontend/myapp #Promena foldera
npm install #Instalacija svih zavisnosti(nalaze se u package.json fajlu)
npm start #Pokretanje aplikacije
```

Aplikacija je dostupna na http://localhost:3000
