# Mobilna Aplikacija

## Kratak opis

- Kompatibilna je i sa androidom i sa ios-om.
- Omogucava korisnicima da vide sve knjige, kao i da obrisu knjigu.
- Aplikacija komunicira sa backendom iz ASP .net Core.

## Tehnologije

- **React Native** - za izradu mobilnih aplikacija(koju su kompatibilne i sa androidom i sa ios-om).
- **Expo** - za lakse testiranje

## Instalacija i pokretanje

```bash
git clone http://gitlab.pmf.kg.ac.rs/si2025/vexa.git #Kloniranje repozitorijuma
cd sandbox/Marko_Mladenovic/demo/mobile/myapp #Promena foldera
npm install #Instalacija svih zavisnosti(nalaze se u package.json fajlu)
npx expo start #Pokretanje aplikacije
```

Moguce je pristupiti putem telefona, potrebno je preuzeti Expo Go aplikaciju https://expo.dev/go 
Na web-u je dostupna na: http://localhost:8081
