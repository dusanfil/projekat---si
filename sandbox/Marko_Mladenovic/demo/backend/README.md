# BACKEND Aplikacije

## Kratak opis

Predstavlja backend za aplikaciju.

## Tehnologije

- **C# ASP .NET Core** - https://dotnet.microsoft.com/
- **MongoDB** - https://www.mongodb.com/ - NoSQL baza.

## Instalacija i pokretanje aplikacije

```bash
git clone git@gitlab.pmf.kg.ac.rs:si2025/vexa.git
cd sandbox/Marko_Mladenovic/demo/backend/WebApplication1
dotnet restore #Restore svih paketa, tj. zavisnosti(dependency)
dotnet build #Provera da li je sve ok
dotnet run #Pokretanje aplikacije
```
Api rute su vidljive na: http://localhost:5246/swagger
